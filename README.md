# 点字グループトーク

<img width="1200" height="630" alt="OGP" src="https://github.com/user-attachments/assets/e26a8d68-db21-49fd-9e0d-312164ad9879" />


## 概要

点字グループトークというサービスは、点訳サークルなどで点字を学習している人々のための、一言点字クイズを送り合うことができる招待制掲示板サービスです。

### 特徴

利用ユーザーは掲示板を作成することができ、やりとりに一言点字を付与することができます。<br>
一言点字は凸面・凹面で表示され、ひらがなを正解ボタンで見ることができます。<br>
ユーザー同士のやり取りの中で、クイズ形式によって頻繁に点字に触れることにより、ユーザーはより早く・楽しく点字を覚えることができます。

## URL

https://tenji-grouptalk.net/

## 使い方

### グループを作成し、メンバーを招待する
「グループを作成する」ボタンを押し、グループを作成してください。グループは複数作成できます。

「メッセージを作成」ボタンを押すとメッセージが作成されるので、招待したい人に送ってください。

### トーク、コメントを作成し、点字クイズをつける
グループ内では掲示板にあたるトークを作成できます。<br>
また、トークにはコメントつけることができます。<br>

トーク、コメントにはそれぞれ点字クイズ用のひらがな欄があります。<br>
クイズにしたいひらがなをそこに入力します。<br>
<img width="574" height="615" alt="スクリーンショット 2025-12-10 16 06 04" src="https://github.com/user-attachments/assets/7f03a289-4cc5-4762-b185-66f5398d93f2" />

### やり取りの中で点字クイズを解く
点字クイズは凸面・凹面の両方で表示され、ひらがなは隠されています。<br>
やり取りの中で点字の対応表や法則性に触れていき、点字を覚えてみましょう。<br>
